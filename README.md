# Minha Jornada

Aplicativo local de planejamento pessoal, checklist, dashboards e finanças.

## Abrir

Clique duas vezes em `abrir_app.cmd` ou abra `index.html` em um navegador moderno.
Os dados ficam salvos localmente no navegador.

Para usar em mais de um dispositivo, utilize **Dados e backup > Baixar backup** e importe o arquivo no outro dispositivo.

## Usar no celular

### Na mesma rede Wi-Fi

1. No computador, abra `usar_no_celular_mesma_rede.cmd`.
2. Veja o endereço IPv4 mostrado na janela.
3. No celular, abra `http://ENDERECO-IP:4173`.
4. Mantenha a janela aberta durante o uso.

### Uso permanente

Publique esta pasta em uma hospedagem estática com HTTPS. Depois, use **Instalar aplicativo**
ou **Adicionar à Tela de Início** no navegador do celular.
